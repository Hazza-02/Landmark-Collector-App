# Landmark-Collector-App
An app where the user can take photos of landmarks as they are travelling and can collect them.

- A cross-platform mobile app (React Native + Expo) using FastAPI for the backend and Supabase (PostgreSQL) for the database. 

- The app allows the user to take photos of landmarks around the world when travelling to collect them. 

- The aim of this project is to ship a fully functioning mobile app.
